# Optional Web Demo

This folder is reserved for a future lightweight interface after the package and CLI are stable.

## Why it is separate

- the paper's primary artifact should remain the installable software package
- web code changes faster than the scientific core
- keeping UI code isolated prevents deployment concerns from muddying the core reproducibility story

## Suggested future scope

- FASTA upload and validation
- motif annotation preview
- tissue-target selection for candidate design
- downloadable CSV and JSON results
