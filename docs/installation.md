# Installation

## Local editable install

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e .
```

## Verification

```bash
tomato-promoter-designer validate-input --input examples/demo_input.fasta
```

## Future release targets

- `pip install tomato-promoter-designer`
- `conda install -c <channel> tomato-promoter-designer`
